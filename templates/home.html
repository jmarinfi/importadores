Instrucciones Importador
  1. Elegir, en la barra superior, el tipo de instrumento asociado a los datos que se desean importar.
  2. En algunos casos, es posible introducir los datos a partir de un formulario y de un fichero Excel con formato pre-fijado. En otros casos, sólo es posible hacerlo con el fichero Excel de formato pre-fijado.
  3. En cada una de las secciones, se ofrece una plantilla Excel con el formato pre-fijado de importación de datos.
  4. El importador muestra el resultado de la importación, junto con el histórico.
  5. El resultado se puede aceptar o guardar para aceptarlo más adelante.
  6. Al aceptar, el importador devuelve el archivo CSV para importar a Tunneldata.
  7. Adicionalmente, el importador devuelve un archivo resumen de la importación para su archivo.
